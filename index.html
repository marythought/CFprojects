Rabbit, Run

JavaScript, HTML

How fast and focused do you need to be to race Peter Rabbit? Click the rabbit to play and find out! This game uses JS objects and functions ("this.speed" and "this.position") and prints results to the console. Second week assignment for Code Fellows Foundations I.

Bletchley Bot

Python & Twitter

More than 75% of Bletchley Park's code-cracking veterans are women. Using Python, I created a bot that tweets their names and service summaries every 30 minutes. Through this process I practiced using Python to manipulate .csv data and communicate with twitter's API. I also learned to prep and deploy apps using git and Heroku.

Number Guessing Game

JavaScript

Player guesses a secret number between 1 and 100, the game keeps track of your high and low to narrow the range and reports back when you've won with how many guesses it took. Uses 'prompt' and 'alert' to interact with player. Simple little JS game, first assignment of Code Fellows Foundations I class.
